# Usmania Academy - Programming Tutorials Website

A modern, responsive website for Usmania Academy's YouTube programming tutorials built with React, Vite, and Tailwind CSS.

## 🚀 Features

- 📱 Fully responsive design
- 🎥 YouTube video integration
- 🔍 Video search functionality
- 📚 Course categories
- 💬 Related video suggestions
- 🌓 Dark theme optimized
- ⚡ Fast performance with Vite
- 🎨 Modern UI with Tailwind CSS

## 🛠️ Tech Stack

- **React 18** - UI Framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router DOM** - Navigation
- **YouTube Data API v3** - Video fetching
- **React Icons** - Icons

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/usmansyedcoder/UsmaniaAcademy.git
cd UsmaniaAcademy
Install dependencies:

bash
npm install
Create a .env file in the root directory:

env
VITE_YOUTUBE_API_KEY=your_youtube_api_key_here
VITE_CHANNEL_ID=your_channel_id_here
Start the development server:

bash
npm run dev
🚀 Build for Production
bash
npm run build
npm run preview
